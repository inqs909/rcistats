The goal of `rcistats` is to provide helper functions for courses offered at California State University Channel Islands.

## Installation

You can install the development version of csucistats from
[GitHub](https://github.com/inqs909/rcistats) with:

``` r
install.packages('rcistats', 
                 repos = c('https://inqs909.r-universe.dev', 
                           'https://cloud.r-project.org'))
```

You may also install this package to be used in quarto webr:

    webr::install('rcistats',
                  repos = c('https://inqs909.r-universe.dev', 
                            'https://repo.r-wasm.org'))
